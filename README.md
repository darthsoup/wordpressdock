# wordpressdock

My quick and dirty wordpress docker container.

Inspired by [laradock](https://github.com/laradock/laradock).

## Installation

Download and run the following command

```sh
docker-compose up -d wordpress-php71
```
Supported PHP Versions:

 - 7.2 `wordpress-php72`
 - 7.1 `wordpress-php71`
 - 7.0 `wordpress-php70`
 - 5.6 `wordpress-php56`
